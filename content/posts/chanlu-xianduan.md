# 什么是线段？

线段是比笔更大的级别，由多笔组成。

## 线段的定义

**线段** = 至少3笔构成的趋势 + 被反向笔破坏

```
上涨线段：至少3笔向上
下跌线段：至少3笔向下
```

## 线段的特征

1. **稳定性**：比笔更稳定，不会轻易被破坏
2. **方向性**：有明确的上涨或下跌方向
3. **级别**：比笔高一个级别

## 线段被破坏

线段会被反向的运动破坏，这是判断趋势转折的重要信号。
