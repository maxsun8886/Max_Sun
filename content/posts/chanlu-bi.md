# 什么是笔？

笔是缠论中最基础的结构单元。

## 笔的定义

**笔** = 至少5根连续的K线 + 1个独立的高低点

```
上涨笔：低点不断抬高
下跌笔：高点不断降低
```

## 笔的构成

- **起始点**：笔的起点
- **结束点**：笔的终点
- **中间K线**：至少3根

## 笔的意义

笔代表了股价的最小级别的运动趋势，是构建更大级别结构的基础。

学会画笔，是缠论学习的第一步 📈
